# Smart AI Chatbot (Mental Health + Legal Assistant)

## Overview
Starter project using:
- Streamlit UI
- SQLite (dev) persistence
- SentenceTransformers for embeddings
- Hugging Face pipeline for sentiment
- FAISS for local retrieval (supports .txt and .pdf)
- OpenAI stubs for LLM responses
